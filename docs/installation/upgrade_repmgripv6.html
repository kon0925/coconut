Replication Manager IPv6 Support

New Function

Replication Manager now supports the IPv6 internet protocol and creates IPv6 connections whenever possible. If IPv6 is not available, Replication Manager creates IPv4 connections instead. Applications can optionally supply a socket approval callback for additional control over the connections created by Replication Manager.

New Function

  • DB_ENV->repmgr_set_socket()
